# 🌾 AgroTech Monitoramento

Projeto interativo que une **tecnologia e bem-estar no agro**, mostrando como inovação pode ajudar agricultores a monitorar plantas daninhas, insetos e aplicar práticas sustentáveis.

## 🚀 Funcionalidades

- 📡 **Processamento de imagens de drones**  
  Captura dados detalhados da lavoura para análise precisa.

- 🗺️ **Mapas geoespaciais de alta precisão**  
  Auxiliam no plantio e colheita com maior eficiência.

- 🌱 **Monitoramento de plantas daninhas e falhas de plantio**  
  Identificação rápida de problemas no campo.

- 💧 **Aplicação de insumos em taxa variável**  
  Reduz desperdícios e aumenta a produtividade.

- 🐞 **Detecção de insetos e pragas**  
  Área interativa com 5 plantas e insetos que podem ser eliminados ao clicar.

- 🎯 **Quiz AgroTech**  
  Quiz com 5 perguntas sobre tecnologia e agro, liberado apenas após ativar todos os botões de monitoramento.

## 🖼️ Visual

- Fundo geral com imagem personalizada (`fundo.jpg`).  
- Quiz destacado com fundo próprio (`quiz-fundo.jpg`).  
- Estilo moderno com cores **amarelo, azul e verde**, hover criativo e sombras.

## 📂 Estrutura

- `index.html` → Estrutura principal da página.  
- `style.css` → Estilo visual, fundos e layout.  
- `main.js` → Lógica interativa (monitoramento, contador, quiz).  
- `fundo.jpg` → Imagem de fundo geral.  
- `quiz-fundo.jpg` → Imagem de fundo exclusiva do quiz.

## ▶️ Como rodar

1. Baixe/clique os arquivos em uma mesma pasta.  
2. Abra o `index.html` em qualquer navegador.  
3. Clique nos botões de monitoramento para liberar o quiz.  
4. Interaja com as plantas e insetos para simular o controle de pragas.  
5. Responda ao quiz e veja sua pontuação final.

## 🌍 Objetivo

Demonstrar como **tecnologia e agro** podem andar juntos em harmonia, trazendo eficiência, sustentabilidade e inovação para o campo.
